How to Find Your Own IP Address
Auto-detection isn't reliable across all network setups, so find your IP manually and whitelist it below — this guarantees ThreatMon won't flag your own device.
  1. Open Command Prompt (press Win + R, type cmd, hit Enter).
  2. Type ipconfig and press Enter.
  3. Find the network adapter you're actually using (usually Wireless LAN adapter Wi-Fi), and look for the line that says IPv4 Address.
  4. Copy that IP address (e.g. 192.168.100.31) and paste it into the form below.
Ignore adapters labeled VirtualBox, VMware, or Bluetooth — those aren't the IP your real network traffic uses.
